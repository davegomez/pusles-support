# Pusles Support

Support and contact for **Pusles** — a free, offline collection of classic puzzle games (Sudoku, Memory Match, Sliding Puzzle, Tic-Tac-Toe) for iOS and Android.

## Get help

- **Found a bug or have a question?** [Open an issue](https://github.com/davegomez/pusles-support/issues/new) — please include your device, OS version, and what happened.

## Privacy

Pusles collects no personal data. All game progress stays on your device. The app collects only anonymous diagnostics (crash, performance, and usage metrics) that cannot be traced back to you.

Full policy: [www.pusles.app/privacy](https://www.pusles.app/privacy)
